프로젝트 개요

서울대학교 기계공학부 AI주도 시뮬레이션 및 설계 연구실의 특허 기술(딥러닝 기반 SEM 이미지 분석)을 실용화하기 위한 MLOps Desktop App 개발 프로젝트입니다.

연구원들의 SEM 이미지 데이터셋 관리 및 라벨링 등의 전처리, AI 모델 학습, 추론 결과 시각적 조정 등 전 과정을 하나의 플랫폼에서 효율적으로 빠르게 수행하도록 하는 것이 목표입니다.

마스킹/정렬 등 복잡한 편집 기능을 안정적으로 제공하기 위해 Electron 기반 데스크톱 환경 위에 React로 프론트엔드를 구성하고, Python·FastAPI·PyTorch로 백엔드를 구성하는 풀스택 아키텍처를 채택했습니다.

UNet 기반 Segmentation 모델 통합, 실시간 GPU 모니터링, 비동기 학습 프로세스 관리, Server-Sent Events 스트리밍 등 다양한 기술을 활용하였습니다.

시스템 아키텍처

주요 기능

01

데이터셋 관리 (Dataset Management)

📂 핵심 기능

  • 다양한 형식 지원: SEM 이미지(PNG, JPG, TIFF), 어노테이션 데이터(JSON, CSV) 등 다양한 파일 형식 지원
  • 효율적인 파일 관리: 드래그 앤 드롭으로 쉽게 업로드, 대량 파일 일괄 처리 가능
  • 스마트 분류 시스템: Detection/Segmentation/Alignment 등 용도별 자동 분류
  • 썸네일 미리보기: 실시간 썸네일 생성으로 빠른 이미지 브라우징
  • 어노테이션 상태 추적: 각 파일의 어노테이션 완료 여부를 한눈에 확인

🗄️ 데이터 구조 관리

  • Train/Valid/Test 분할: 머신러닝 학습을 위한 데이터셋 자동 분할
  • 클래스 정보 관리: 데이터셋별 독립적인 라벨/클래스 체계 구성
  • 메타데이터 관리: 이미지 해상도, 생성일, 수정일 등 상세 정보 저장

📊 고급 기능

  • 일괄 내보내기: 선택한 파일들을 ZIP 형식으로 압축하여 다운로드
  • 버전 관리: 데이터셋 버전 히스토리 추적
  • 검색 및 필터링: 파일명, 타입, 어노테이션 상태 등으로 빠른 검색
02

Annotation Tool - Segmentation/Detection

🖌️ 다양한 어노테이션 도구

  • Brush Tool: 자유롭게 그리며 픽셀 단위 정밀 마킹 - 브러시 크기 조절 가능, 실시간 미리보기
  • Polygon Tool: 클릭으로 점을 찍어 다각형 영역 지정 - 정밀한 경계선 표현, 점 추가/삭제 편집 기능
  • Bounding Box: 객체 탐지를 위한 사각형 영역 설정 - 빠른 라벨링 작업, 크기 및 위치 실시간 조정

🤖 AI 지원 기능

  • AI Assist: 학습된 모델을 활용한 자동 세그멘테이션 - 초기 어노테이션 자동 생성, 수동 보정으로 정확도 향상
  • Smart Selection: 유사한 패턴 자동 인식 및 선택

📊 실시간 분석

  • 면적 계산: 선택 영역의 픽셀 수 및 실제 면적 자동 계산
  • 통계 정보: 클래스별 객체 수, 전체 커버리지 비율 표시
  • 어노테이션 디테일: 각 객체의 위치, 크기, confidence 점수 표시

⚡ 편의 기능

  • Undo/Redo: 무제한 작업 히스토리 관리
  • 줌 & 팬: 고해상도 이미지의 세밀한 작업 지원
  • 단축키 지원: 빠른 도구 전환 및 작업 효율성 향상
03

Annotation Tool - Image Alignment (이미지 정렬)

🎯 이미지 정렬 도구

  • 듀얼 뷰어: 두 개의 SEM 이미지를 동시에 표시하여 비교 분석
  • 대응점 매칭: 클릭으로 두 이미지의 대응점 지정 - 최소 3개 점으로 자동 정렬, 색상 구분으로 쉽게 식별

🔄 변환 기능

  • 실시간 변환: 회전, 이동, 크기 조절을 실시간으로 적용 - Shift키: 회전 모드, Ctrl키: 스케일 모드, 드래그: 이동 모드
  • 투명도 조절: 오버레이 투명도 조정으로 정렬 정확도 확인
  • 변환 행렬 계산: Affine/Perspective 변환 자동 계산

✨ 애니메이션 효과

  • 부드러운 전환: 변환 적용 시 자연스러운 애니메이션
  • 토글 뷰: 정렬 전후 비교를 위한 빠른 전환
  • 히스토리 관리: 정렬 작업 이력 저장 및 복원
04

Task - 모델 학습 및 추론 (Training & Inference)

🖥️ GPU 리소스 관리

  • 실시간 모니터링: GPU 사용률, 메모리, 온도 실시간 표시
  • 멀티 GPU 지원: 여러 GPU 중 선택하여 학습 진행
  • 자동 최적화: GPU 메모리에 맞춰 배치 크기 자동 조정

⚙️ 학습 설정

  • 하이퍼파라미터 튜닝: Epoch 수, 학습률(Learning Rate), Batch Size, Optimizer 선택, Loss Function 설정
  • 데이터 증강: 회전, 플립, 노이즈 추가 등 자동 증강
  • 모델 아키텍처: U-Net, Mask R-CNN 등 다양한 모델 지원

📈 학습 진행 관리

  • 실시간 로그: 학습 진행 상황 실시간 스트리밍
  • 성능 지표 시각화: Loss, Accuracy, IoU 등 차트로 표시
  • 체크포인트 저장: 주기적 모델 저장 및 베스트 모델 자동 선택
  • 조기 종료: 성능 향상이 없을 시 자동 학습 중단

🚀 모델 추론

  • 모델 관리: 체크포인트 선택, 모델 정보 표시 (학습 데이터셋, 성능 지표, 파라미터 수)
  • 배치 추론: 대량 이미지 처리, 진행률 표시, GPU를 활용한 병렬 처리
  • 결과 시각화: 오버레이 뷰, 사이드 바이 사이드 비교, 투명도 조절, 다양한 컬러맵
  • 결과 활용: 어노테이션 자동 변환, 통계 분석, PNG/JSON/CSV 내보내기
